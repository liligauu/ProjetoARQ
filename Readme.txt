Abra um terminal na pasta do projeto.
Execute o seguinte comando:

python mips.py

Funções do simulador:
Carregar .o: Selecione um arquivo .o com instruções MIPS para simular.
Executar Próxima: Executa a próxima instrução da lista.
Executar Tudo: Executa todas as instruções até o fim.
Limpar: Reinicia o simulador e limpa o terminal e registradores.

Observações:

Registradores são exibidos no formato: R[num] [nome] = valor.
A interface inclui terminal e painel de registradores lado a lado.
